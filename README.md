>> <h3> Resume </h3>
-----
<p> TThis is a shell script for Ubuntu systems. It works for both Linux natives and WSL users. Here we make an alias for the python executors, giving the user the choice of which version of python he can choose to change his executor.
</p>

>> <h3> How install </h3>

1. Make sure you have Bash installed on your machine.

2. Download the ```alias_pyVersion.sh``` file to your machine.

4. Give the ```alias_pyVersion.sh``` file execute permissions using the following command: ```chmod +x alias_pyVersion.sh```.

5. Run the script using the following command: ```source alias_pyVersion.sh```.
